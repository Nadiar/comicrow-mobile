# Contributing to ComicRow

Thanks for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork and install Flutter (3.32+)
3. Run `flutter pub get` to install dependencies
4. Make your changes on a new branch

## Before Submitting

Make sure your changes pass:

```bash
flutter analyze    # Zero issues required
flutter test       # All tests must pass
```

If you're adding new functionality, include tests.

## Pull Requests

- Keep PRs focused — one feature or fix per PR
- Write a clear description of what changed and why
- Reference any related issues

## Reporting Bugs

Open an issue with:

- Steps to reproduce
- Expected vs actual behavior
- Device/OS info if relevant

## Code Style

- Follow existing patterns in the codebase
- Run `flutter analyze` before committing
- Dart formatting is enforced by `dart format`
